# edds2image

High-performance CLI tool to convert `.edds` files into **PNG**, **TIFF**, and **DDS**.

## How to Use
- **Batch:** Place in a folder with `.edds` files and run.
- **Drag & Drop:** Drop `.edds` files directly onto the EXE.
- **Association:** Right-click an `.edds` > *Open with* > `edds2image.exe`. Double-click to convert all files in that folder.
- **Manual:** Run in an empty folder to open a File Selection Dialog.

## Output
Files are saved into `/png`, `/tif`, and `/dds` subfolders.

## Requirements
- Windows 10/11 (64-bit)
- .NET 8.0 Desktop Runtime
